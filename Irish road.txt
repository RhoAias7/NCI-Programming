You have been commissed by Irish Road to develop an application which will assist with calculating the total amount due in M50 tolls at the end of each month.
  
The app should ask the user to input the number of single trips and return trips undertaken during the month.  It should then use the instantiable class to calculate the total cost and output this figure to the user.

Please note that single trips cost €3 and return trips cost €5.  

Input

2 questions - 2 setter, 2 inputs

Process

(noOfSingleTrip*singleTripPrice)+(noOfReturnTrip*returnTripPrice)

Output

1 output, 1 getter, 1 setter
